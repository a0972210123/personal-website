## 0. 本週地圖

**驅動問題：** Netflix 廢掉打卡、休假規定與報銷審批，績效反而上升——這套「自由與責任」的前提是什麼，抄襲它的公司死在哪？

1. 說明 Netflix 文化的邏輯鏈：人才密度 → 完全坦誠 → 拆除控制 → 情境領導

3. 對比高控制與高自由兩種文化策略，判斷什麼組織該用哪種

## 1. 課文

### 1.1 那份簡報：矽谷最重要的文件

Patty McCord（Netflix 前 CHRO）與 Reed Hastings 打造的「Netflix Culture Deck」被 Sheryl Sandberg 稱為「矽谷有史以來最重要的文件」。它的力量不在口號，在一條**順序不能亂的邏輯鏈**——多數模仿者死於只抄了鏈條的後半。

### 1.2 四步邏輯鏈（順序即是全部）

**第一步：人才密度（talent density）**
地基假設：**最好的工作環境，是身邊全是一流同事**——這是激勵因子不是保健因子（W40：頂尖的人渴望與頂尖的人共事，勝過渴望按摩椅）。反向的病：一個平庸者的傷害不是「他那份產出平庸」，是他**拉低整個房間的標準與節奏**（一流者開始降速遷就，或離開）。所以 Netflix 付**市場頂薪**、只留 A 級人才——密度是後面一切的前提。

**第二步：完全坦誠（radical candor / honesty）**
密度夠高，才承受得起第二步：回饋**直接、即時、對事**——「只在當面說你敢在背後說的話」、正式的 360 回饋用真名。**順序警告**：在低密度團隊推坦誠=互相傷害大會（能力不足的人收到的全是負回饋，防衛與政治接管一切）；高密度團隊的坦誠=高手互相切磋（W41 學習區的文化版：對事的標準狠、對人的安全高——密度讓「狠」不毀人，因為人人接得住）。

**第三步：拆除控制（remove controls）**
密度＋坦誠到位，控制變成冗餘：Netflix 廢掉休假規定（「自己判斷」）、差旅報銷政策只有一句——「**以 Netflix 的最大利益行事**」。邏輯：對成年人，流程是把 3% 壞人的防範成本攤在 97% 好人頭上（每個審批都在對好人說「我們不信任你」——保健因子引爆＋速度歸零）。**與 W29 Deming 的和解**：Deming 說改系統防錯、Netflix 說拆系統信人——場景之別：**錯誤成本結構**決定一切。創意工作的錯誤可逆（一部爛片、一個爛功能，改就是了）→ 控制的成本（慢、羞辱）> 錯誤的成本 → 拆；製造/醫療/金融的錯誤不可逆 → 規格化與防呆是對的（W28 的規則不是官僚）。

**第四步：情境領導（context, not control）**
控制拆掉後管理靠什麼？**把情境講透**：策略、市場、限制、優先序（W4 OKR 的透明、W26 備忘錄的清晰）——然後讓貼近現場的人自己做對決策。主管的失職不再是「沒管好」，是「情境沒給夠」——下屬做了蠢決策，第一個問題是「我哪個 context 沒講清楚？」（W1 你的產出=組織的產出、W29 的系統責任，同一條血脈）。

### 1.3 維持密度的機制：留任者測試

> **Keeper test**：「如果這個人明天提離職，我會不會拚命留他？」——答案是「不會」，就該給豐厚遣散費、好聚好散。

配套的名句：「**adequate performance gets a generous severance**」（表現夠好=體面送別）——Netflix 把團隊比作**職業球隊而非家庭**：家庭無條件接納，球隊按位置擇優——這是全套系統中最受爭議的一刀。

**誠實的批判視角**（不吞下整套教條）：
- **前提昂貴**：頂薪＋極挑的招聘（W42 的整套嚴選）＋錯誤可逆的行業——三者缺一，抄這套=災難
- **代價真實**：常態性的不安全感（keeper test 焦慮）與 W41 心理安全感存在張力——Netflix 的辯護是「對績效的高標準」與「對發言的安全」是兩軸（正是 Edmondson 的 2×2），但執行走樣時容易雙輸
- **模仿者的死法**：只抄第三步（無限假、no rules 的酷）而不抄第一步（頂薪嚴選）與第二步（坦誠回饋）——低密度＋高自由＝放牛吃草，鏈條斷在第一環

### 1.4 文化的操作性定義（本週最重要的一句）

> **文化不是牆上的價值觀海報，是你「獎勵誰、容忍誰、請走誰」的累積紀錄。**

員工不讀海報，員工讀案例：誰被升了（原來那樣會贏）、誰的惡行被容忍了（原來那可以）、誰被請走了（原來那是紅線）。推論三條：
1. 文化從**第一號員工**就開始寫，且主要由**例外處理**寫成（順風時人人像海報，文化在艱難決定裡現形——W22 深植文化的機制）
2. **明星員工的惡行是文化的頭號殺手**：容忍「業績好但有毒」的人=公告「價值觀在業績前讓路」——一次讓路，海報全毀（W42 Q5 的致命傷邏輯）
3. 價值觀要配**代價**才可信（W8 文化品牌的內部版）：寫「誠實」就要獎勵報壞消息的人（W41 頭三秒）、寫「顧客至上」就要有為顧客虧錢的案例

## 2. 原典導讀

### 📄 必讀一：McCord,〈How Netflix Reinvented HR〉(HBR, 2014)
🔗 https://hbr.org/2014/01/how-netflix-reinvented-hr （需訂閱／有免費額度）

**讀法（約 35 分）：** McCord 親筆的第一手總結：廢除休假政策與績效改善計畫（PIP）的始末、「只雇成年人」的完整論證、以及她最鋒利的一句——「最好的福利是優秀的同事」。**帶著問題讀：** 她說多數 HR 流程是「為了防最差的 3% 而設計」——你現在公司的哪三個流程符合這個描述？拆掉的前提各是什麼？

### 📄 必讀二：Netflix 官方文化頁（Culture Deck 的現行版）
🔗 https://jobs.netflix.com/culture （免費）

**讀法（約 30 分）：** 對照 2009 簡報的網路存檔與現行版的措辭演化（例如 keeper test 的表述趨於柔和）——文化文件本身也在迭代。注意它的文體：不是形容詞清單，是**行為描述＋真實的取捨**（我們選擇 X 放棄 Y）——這正是 W26「可反駁性」在文化文件上的應用：寫不出放棄什麼的文化宣言，是海報不是文化。

**延伸（選讀）：** Hastings & Meyer《零規則》（天下雜誌）——CEO 版全套故事，含國際化時的文化衝突章節（坦誠在日本與荷蘭的不同校準——文化策略的在地化）。

## 3. 實作任務（50 分）

**寫你未來公司的「文化一頁」草稿：**

1. **三條真實的價值觀**——每條必須配一個「**我們因此會做的難決定**」實例（句型：「我們重視 X，所以當 Y 情境發生時，我們會選 Z——即使代價是 W」）。寫不出代價的那條，刪掉重想（沒有代價的價值觀是口號）
2. **判斷你的行業座標**：你的創業題目屬於錯誤可逆（創意/軟體）還是不可逆（安全關鍵）？據此決定：哪些環節高自由（拆控制）、哪些環節高控制（規格化＋防呆，W28/W29）——寫成一張兩欄表
3. **第一個例外預演**：預想「業績最好的員工違反第一條價值觀」的場景——你的處理劇本是什麼？（這題的答案才是你真正的文化）
4. **keeper test 自問**：把它反轉用在自己身上——「我現在做的事，一年後的我會拚命挽留嗎？」（W23 個人雙元、W51 的預熱）
5. 存檔——與 W40 職缺、W42 選才包合併為「人才系統 v0.2」（W52 復盤收攏）

## 4. Quiz

**Q1（單選）** Netflix 邏輯鏈的正確順序是：
- A. 拆除控制 → 人才密度 → 坦誠 → 情境
- B. 人才密度 → 完全坦誠 → 拆除控制 → 情境領導——每步是下一步的前提
- C. 情境領導 → 坦誠 → 密度 → 拆控制
- D. 四者可以任意順序推行

**Q2（單選）** 「只抄無限休假、不抄頂薪嚴選與坦誠回饋」的公司，最可能的結局是：
- A. 成為下一個 Netflix
- B. 低密度＋高自由＝標準渙散、搭便車者橫行——鏈條斷在第一環
- C. 員工滿意度持續提升
- D. 沒有影響，休假政策與績效無關

**Q3（單選）** 「文化是獎勵誰、容忍誰、請走誰的紀錄」——依此，摧毀文化最快的單一事件是：
- A. 取消年度尾牙
- B. 容忍業績最好的員工公然違反核心價值觀——向全公司廣播「價值觀可以讓路」
- C. 辦公室搬遷
- D. 更換企業識別標誌

**Q4（簡答）** keeper test 與心理安全感（W41）的張力何在？Netflix 的辯護是什麼？你認為這個辯護在什麼執行條件下成立、什麼條件下破產？

**Q5（簡答）** 「錯誤成本結構決定控制水位」——用這個原則替一家「軟體新創但處理醫療數據」的公司設計文化：哪些環節學 Netflix、哪些環節學豐田（W28）？

<summary>👉 展開解答與解說</summary>

**Q1：B。** 鏈條的因果：沒有密度，坦誠變成互相傷害；沒有坦誠，拆控制後無人糾偏；沒有拆控制，情境領導無用武之地（都被流程管死了）。每一步都在為下一步「掙得資格」——這也是為什麼它難抄：後面的步驟看得見（no rules 很酷），前面的投資看不見（頂薪與嚴選很貴）。

**Q2：B。** 自由的前提是「值得被信任的人」＋「敢互相糾正的文化」。缺了前兩環，自由=無人管理的公地：認真的人替散漫的人擦屁股、然後心冷離開——逆淘汰啟動（好人走、混子留），密度進一步下降，死亡螺旋。

**Q3：B。** 例外事件的廣播效應遠大於政策文件：一次「他業績好，算了」，等於把價值觀標價（低於一個 top sales 的業績）。全公司立刻完成學習：海報是裝飾，業績是聖旨。這也是為什麼 1.4 說文化主要由「例外處理」寫成。

**Q4 參考答案：** 張力：keeper test 製造「位子隨時可能不保」的生存焦慮，而焦慮正是心理安全感的反面——怕被踢出球隊的人，敢報壞消息、敢認錯嗎？Netflix 的辯護：兩者是不同軸（Edmondson 2×2）——keeper test 針對「績效標準」（你夠不夠好），心理安全感針對「人際風險」（你敢不敢說）；理論上可以同時高。成立條件：①標準透明且對事（被請走的人知道為什麼，且不是因為說了真話）；②「說真話、認錯、報壞消息」被明確地**排除在 keeper test 的扣分項之外**、甚至是加分項（頭三秒文化，W41）；③遣散真的體面（降低出局的災難感）。破產條件：主管把 keeper test 當威嚇工具、或用它處決唱反調者——那一刻兩軸合併，全員進入焦慮區（W41），假數據時代開始（W29）。

**Q5 參考答案（示例）：** 分軸設計。**學 Netflix 的環節**（錯誤可逆、判斷密集）：產品設計與迭代（快速實驗，W19）、行銷內容、內部協作（報銷/休假從簡、資訊透明、坦誠回饋）——這裡的敵人是「慢」。**學豐田的環節**（錯誤不可逆、合規密集）：病歷資料的存取（最小權限＋稽核軌跡，W34）、資料處理管線的變更（標準作業＋審查＋防呆——任何人「自己判斷」直接改生產資料庫是災難）、模型輸出進入臨床流程前的驗證關卡（W33 的人為把關）——這裡的敵人是「錯」。文化一頁要明寫這條線：「我們在產品上開快車、在病人資料上開火車——兩套規則，一個理由：錯誤成本。」並用招聘與案例讓兩套並存不精神分裂（工程師知道何時切換模式——正是 W23 雙元的個人版：同一批人、兩種作業系統、清楚的切換邊界）。
