## 1. 暖身：Q1 十二週的一句話地圖

先閉卷自測：每週用一句話說出「它給了我什麼武器」，說不出來的就是洞。對照版：

| 週 | 武器 | 一句話 |
|---|---|---|
| W1 | 效能 vs 效率 | 先做對的事，你的產出=組織的產出 |
| W2 | 決策防呆 | 偏誤擋不住，用流程（premortem、清單）擋 |
| W3 | 經濟四把刀 | 機會成本、邊際、彈性、誘因——永遠問「相較於什麼」 |
| W4 | OKR | 目標要少、要 outcome、要與考核鬆綁 |
| W5 | JTBD | 顧客僱用產品完成任務，看行為不問偏好 |
| W6 | STP | 行銷=選擇：切市場、打灘頭堡、佔心智位置 |
| W7 | 定價 | 價格是最大利潤槓桿；價值定價；付費心理 |
| W8 | 品牌×漏斗 | 品牌是長期資產＋每層漏斗的折扣券 |
| W9 | 三大報表 | 淨利是意見，現金是事實 |
| W10 | 杜邦 | ROE 必拆三引擎，看獲利的品質 |
| W11 | 單位經濟 | 貢獻邊際、兩平點、LTV/CAC 與回收期 |
| W12 | 計分卡 | 領先配落後、Goodhart 防鑽、儀表板×衝刺 |

## 2. 主任務：組裝一頁商業儀表板（120 分）

**選題**：你最想做的創業題目；沒有就用「把管理 3R 專欄經營成有收入的內容產品」。

**版型**（一頁 A4 或一個 md 檔，五個區塊由上而下）：

```
┌─────────────────────────────────────┐
│ ① 本季 OKR（W4）：1 個 O ＋ 3 條 KR                    │
├────────────────┬────────────────────┤
│ ② 市場面（W5–8）      │ ③ 財務面（W9–12）        │
│ · JTBD 卡（一句話版） │ · 單位經濟表：價格/變動成本/ │
│ · 目標區隔＋灘頭堡     │   貢獻邊際/兩平點           │
│ · 定位陳述（一句）     │ · LTV/CAC 與回收期（估）    │
│ · 漏斗五層現況指標     │ · 現金狀態：跑道幾個月      │
├────────────────┴────────────────────┤
│ ④ 四構面 KPI（W12）：8 個以內，標【領先/落後】         │
├─────────────────────────────────────┤
│ ⑤ 風險區（W2）：三大決策風險 ＋ premortem 一句結論     │
└─────────────────────────────────────┘
```

**組裝規則：**
1. **直接回收前 12 週的實作產出**：W4 的 OKR、W5 的 JTBD 卡、W6 的 STP、W7 的定價、W8 的漏斗表與品牌一頁、W11 的單位經濟、W12 的計分卡——本週的工作是**整合與對齊**，不是重寫
2. **對齊檢查**（本週真正的學習發生在這裡）：
   - 定位陳述（②）與價格帶（③）一致嗎？（W7：定價是定位的財務表達）
   - 漏斗的轉換假設（②）與 CAC 估計（③）兜得攏嗎？
   - KR（①）能被 KPI（④）追蹤嗎？
   - 兩平點（③）在灘頭堡（②）的規模內達得到嗎？
   - **任何一處對不齊，就是你商業構想的裂縫——把它寫進⑤風險區**
3. 數字允許粗估，但每個數字要能回答「這個數怎麼來的」（W10 的敘事–數字紀律）

## 3. 費曼測試（30 分）

找一位聰明但不懂商業的朋友（或請模型扮演），**10 分鐘講完你的儀表板**：
- 講不下去、開始背名詞的段落 → 標記：那是你「眼熟但不懂」的概念
- 對方問倒你的問題 → 記下來，這是最珍貴的輸出
- 判準：對方能複述「這門生意怎麼賺錢、最大的風險是什麼」= 通過

## 6. 自我檢核（復盤版）

1. 儀表板上哪個數字你最沒把握？它需要什麼資料才能變得可信？（這就是 W19 的第一個實驗）
2. 「對齊檢查」抓出了哪條裂縫？它動搖的是定位、定價還是規模假設？
3. Q1 十二週裡，哪個概念**改變了你原本的想法**？（不是「學到最多」——是「推翻了什麼」）

<details>
<summary>👉 檢核參考（給模型帶討論用）</summary>

- 問題 1 的常見答案是 CAC 或留存——正確的下一步不是「再估準一點」，而是設計一個小實驗實測（Q2 的 W19 會正式教）。
- 問題 2 最常見的裂縫：定位喊高端、定價卻不敢高（W7 價格–品質推論沒內化）；或兩平點需要的量遠超灘頭堡規模（生意在數學上不成立——這是最有價值的發現，恭喜在 Excel 上倒閉而不是在現實中）。
- 問題 3 沒有標準答案，但「淨利是意見現金是事實」「行銷=選擇不做誰」「加薪不是激勵因子」是三個最常被回報的世界觀更新。
</details>
