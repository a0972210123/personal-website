> Q4 特色：每個 playbook 都是「整合前三季知識 + 該產業特有打法」的實戰手冊。

## W39-41：SaaS / 數位工具 Playbook

研究方向：
- PLG（Product-Led Growth）：free trial vs freemium、activation metrics、PQL
- Launch 通路組合：Product Hunt（銜接 Q1 M3）+ HN + Reddit + newsletter sponsorship
- Indie hacker 定價：一次性 vs 訂閱、LTD（lifetime deal）的利弊、AppSumo 生態
- SEO for SaaS：programmatic SEO、comparison pages（"X vs Y"）、free tool marketing（Matt 的捕夢網生成器就是 free tool marketing 的實例）
- 案例研究方向：solo founder 成功案例（如 levels.fyi、Plausible、Bannerbear 的 build in public 歷程）

## W42-44：電商 / 手作商品 Playbook

研究方向：
- 平台組合：Etsy SEO、Pinterest（銜接 Q1 M2）、IG Shopping、Shopify vs 平台依賴
- 手作品牌敘事：maker story、工藝過程內容化（銜接 Q1 M5 process video）
- 小紅書種草 → 轉換路徑（台灣/華語市場）
- 季節性行銷日曆：Q4 假期購物季、母親節等關鍵檔期倒推備貨與內容
- 訂價與產品線：入門款 → 核心款 → 客製款階梯
- Matt 直接適用：telaaurealab.com 的捕夢網（數位檔案+實體可能性）

## W45-46：教育 / 知識型產品 Playbook

研究方向：
- 課程 launch 模式：waitlist → beta → evergreen；webinar funnel
- 知識付費階梯：免費內容 → 電子報 → 低價產品 → 旗艦課程
- 平台選擇：自架（Teachable/Podia）vs 平台（Hahow、Udemy）——台灣市場特性
- 教育內容的信任建立：free value first、學員成果展示
- Matt 直接適用：教育家×創業家定位，One More Step 本身就是信任資產

## W47-48：B2B / 專業服務 Playbook

研究方向：

- 顧問型漏斗：內容 → lead magnet → discovery call → proposal
- 案例研究（case study）撰寫框架
- 長銷售週期的 nurture 策略（銜接 Q3 W38 自動化）

## W49-50：品牌定位與敘事

研究方向：
- StoryBrand 框架（顧客是英雄，品牌是嚮導）
- 定位經典：Positioning（Ries & Trout）、April Dunford《Obviously Awesome》
- 定價心理學：anchoring、decoy effect、charm pricing、價值定價 vs 成本定價
- 品牌聲音（brand voice）文件製作
