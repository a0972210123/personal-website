# 四大產業偏好速查

> 產出任何單元網站時，適用的主題請帶入對應產業備註。隨模組進度漸進補充。

## 1. 數位產品 / SaaS / 工具類

**Matt 的關聯**：telaaurealab.com 捕夢網生成器（free tool）、未來 Somnorva

- 核心通路：Product Hunt、Hacker News、Reddit、X build in public、dev.to
- 內容偏好：changelog 文化、公開數據（MRR/用戶數）、技術深度文
- 免費工具本身就是行銷（free tool marketing / engineering as marketing）
- 決策週期短（self-serve），landing page 與 onboarding 是轉換關鍵
- 禁忌：在技術社群硬銷、隱藏價格、假 build in public

## 2. 實體電商 / 手作商品

**Matt 的關聯**：捕夢網 SVG/PNG 設計販售（Gumroad/Etsy 潛力）、可延伸實體

- 核心通路：Pinterest（長尾導流之王）、IG Reels、Etsy 站內 SEO、小紅書（華語）
- 內容偏好：過程影片（process video，天然不露臉）、before/after、材質細節特寫
- 視覺品質 = 信任；photography/styling 是基本功
- 季節性極強：假期檔期倒推 6-8 週備內容
- satisfying/ASMR 剪輯手法在此類目 CP 值最高
- 禁忌：只發成品不發過程、忽略 Pinterest（誤以為它是社交平台，其實是搜尋引擎）

## 3. 教育 / 知識型產品

**Matt 的關聯**：教育家×創業家定位、One More Step 學習文件集、簡化教練經歷

- 核心通路：電子報（資產）、YouTube 長影音（信任）、LinkedIn/Threads（觀點）
- 內容偏好：free value first——教最好的內容免費給，賣的是系統化與陪伴
- 信任週期長：受眾要看你 3-6 個月才會買課
- 學員成果 > 講師資歷
- 台灣特性：Hahow/PressPlay 平台生態、LINE 社群經營
- 禁忌：知識囤積式賣課（「完整內容在課程裡」）、誇大成果

## 4. B2B / 專業服務

**Matt 的關聯**：OT 專業背景 + AI 顧問潛力、跨界者定位

- 核心通路：LinkedIn（絕對主場）、產業 podcast、案例研究、轉介
- 內容偏好：框架與方法論、產業洞察報告、客戶成功故事
- 決策鏈長且多人：內容要同時說服使用者與決策者
- 個人品牌 > 公司品牌（服務業尤其）
- 禁忌：冷訊息轟炸、只發公司動態不發觀點

## 跨產業通用原則

- 平台選擇 = 受眾在哪 × 內容形式擅長什麼 × 能持續多久（三者交集才做）
- 所有產業都在經歷「社群搜尋化」與「AI 引用來源化」——內容要同時餵人與餵 AI
- 一人/小團隊：深耕 1-2 平台 + 一源多用，勝過全平台淺嚐
