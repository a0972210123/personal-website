REFERENCE · three.js / globe.gl (WebGL)

3D globe reference

Archived from the mattye.dev dementia-exposome globe bake-off (d3 won for production on weight grounds; this WebGL version is kept for reuse in other projects). Countries shaded by global PM2.5 or composite modifiable-dementia-risk PAF — educational modelled estimates. Drag / zoom / auto-spin; hover a country.

Boundaries © Natural Earth (public domain) · PM2.5 © ACAG/WashU, CC BY 4.0 · PAF © Livingston 2024 RRs — modelled. Self-contained: needs only globe.gl.min.js + world-globe.geojson in this folder. Serve the folder (e.g. python3 -m http.server) — a plain file:// open is blocked from fetching the geojson by browser CORS.
