## 講義（~2h）

（開頭 10 分鐘：回顧上週教家人時卡住的地方。）

### 1. 一條恆等式統治一切

$$\text{資產} = \text{負債} + \text{股東權益}$$

翻譯：**公司擁有的每一塊錢資源（左邊），都有來源（右邊）——不是借來的（負債），就是股東的（權益）。**

複式簿記（double-entry bookkeeping）的天才之處：每筆交易**同時記兩邊（或同邊一增一減）**，恆等式永遠成立。記錄自帶防錯機制——兩邊不平，一定有地方記錯了。這套系統用了六百年，至今是所有財報的地基。

### 2. 借貸法則：五類科目的增減方向

「借」（Debit, Dr）與「貸」（Credit, Cr）**沒有好壞含義**，純粹是「左欄」與「右欄」的代號。規則只有一張表：

| 科目類別 | 增加記 | 減少記 | 例子 |
|---|---|---|---|
| 資產 | 借（左） | 貸（右） | 現金、存貨、設備、應收帳款 |
| 費用 | 借（左） | 貸（右） | 房租、薪資、折舊 |
| 負債 | 貸（右） | 借（左） | 應付帳款、借款 |
| 權益 | 貸（右） | 借（左） | 股本、保留盈餘 |
| 收入 | 貸（右） | 借（左） | 銷貨收入 |

記法：**「資產費用長在左邊，負債權益收入長在右邊」**——各科目在自己家的方向增加。每筆分錄借方總額＝貸方總額，永遠。

（為什麼費用跟資產同邊？因為費用最終會**減少權益**——把它想成「權益的負項」，方向自然相反於權益。）

### 3. 完整示範：飲料店的第一個月

六筆交易，從開店到月底結帳（金額單位：元）：

**① 老闆投入現金 50 萬開店**
```
借：現金 500,000　　貸：股本 500,000
```
（資產+50萬 ＝ 權益+50萬，兩邊同增）

**② 現金 20 萬買製茶設備**
```
借：設備 200,000　　貸：現金 200,000
```
（資產內部一增一減，總額不變——恆等式照樣平衡）

**③ 賒購原料 5 萬（先拿貨，月結付款）**
```
借：存貨 50,000　　貸：應付帳款 50,000
```
（資產+5萬 ＝ 負債+5萬。**沒付現金也要記**——這就是應計制的入口）

**④ 本月現金銷售 8 萬；同時結轉用掉的原料成本 3 萬**
```
借：現金 80,000　　貸：銷貨收入 80,000
借：銷貨成本 30,000　貸：存貨 30,000
```
（收入與成本**同期配對**——「配合原則」，毛利 5 萬才有意義）

**⑤ 付本月房租 2 萬（現金）**
```
借：租金費用 20,000　貸：現金 20,000
```

**⑥ 月底提列設備折舊（20 萬÷5 年＝年 4 萬，本月 3,333）**
```
借：折舊費用 3,333　　貸：累計折舊 3,333
```
（**沒有任何現金流出**，但設備的耗損是本月真實成本——專欄 #7 說的「折舊是費用但非現金支出」，源頭就是這筆分錄）

**月底驗算**（試算表的精神）：

| 資產 | 金額 | 負債＋權益 | 金額 |
|---|---|---|---|
| 現金（50−20+8−2） | 360,000 | 應付帳款 | 50,000 |
| 設備淨額（20 萬−3,333） | 196,667 | 股本 | 500,000 |
| 存貨（5−3 萬） | 20,000 | 保留盈餘（本月淨利） | 26,667 |
| **合計** | **576,667** | **合計** | **576,667** ✅ |

本月淨利 ＝ 收入 8 萬 − 成本 3 萬 − 房租 2 萬 − 折舊 3,333 ＝ **26,667**，自動流入權益的保留盈餘——這正是專欄 #7 三表勾稽的第一條線，你剛剛親手把它記出來了。

再看現金：淨利 26,667，但現金只從 50 萬變 36 萬（**減少** 14 萬）——因為買了設備 20 萬。「賺錢」和「現金變多」是兩件事，第 7 週會用現金流量表把這件事講完整。

### 4. 應計制：財報的靈魂設定

**應計制（accrual basis）**：收入在「賺到時」認列（交貨／提供服務），費用在「發生時」認列——**與現金何時進出無關**。

由此誕生「四兄弟」科目，全部是時間差的產物：

| 科目 | 情境 | 性質 |
|---|---|---|
| 應收帳款 | 貨已交、錢未收 | 資產（收錢的權利） |
| 應付帳款 | 貨已拿、錢未付 | 負債（付錢的義務） |
| 預收收入 | 錢已收、貨未交 | **負債**（欠客戶東西——不是收入！） |
| 預付費用 | 錢已付、服務未用 | **資產**（例：預付一年保險，逐月轉費用） |

最反直覺的是**預收收入是負債**：健身房收你年費 12,000，當下一毛錢收入都不能認，要逐月認 1,000——因為服務還沒提供，這筆錢是「欠你的服務」。

**為什麼要這麼麻煩？** 因為應計制讓「這段期間的經營成果」可比較：如果用現金制，公司只要月底拖著不付貨款，獲利就能灌水。代價是：應計制創造了操縱空間（提前認收入、費用資本化）——這正是第 8 週紅旗清單要抓的東西。**應計制是更好的地圖，但地圖不是領土；現金流量表就是用來對地圖查核領土的。**

### 5. 常見誤解闢謠

- **「借＝支出、貸＝收入」**——完全不是。借貸只是左右欄位代號，「借：現金」是現金**增加**
- **「有賺錢＝有現金」**——飲料店淨利 +26,667、現金 −140,000，同一個月
- **「會計是給會計師學的」**——看不懂分錄邏輯的人讀財報，就像看不懂音符的人讀樂譜，只能看別人的二手評論

## 教家人腳本（~30m）

**開場（說故事）**：「我們來開一家虛擬飲料店。我出 50 萬，第一個月：買了 20 萬設備、賒了 5 萬原料、賣了 8 萬、付了 2 萬房租。你覺得這個月是賺還是賠？」

**第一個驚訝**：「帳面上賺了 26,667。但現金從 50 萬變 36 萬——少了 14 萬。賺錢的店，現金卻變少，因為錢變成了設備。**『賺不賺』和『現金多不多』是兩本帳**，這就是為什麼上市公司要交三張報表。」

**第二個驚訝（預收收入）**：「健身房收你 12,000 年費，你猜他們當天能算多少『收入』？」（答案：0——服務還沒給，那是欠你的債。順帶一提：這也是為什麼預售屋、儲值卡商家倒閉時消費者是債權人。）

**互動**：讓家人記一筆「家庭分錄」——「這個月我們付了 3 萬房租」怎麼記？（借：房租費用／貸：現金——恭喜家人寫出人生第一筆分錄。）

**預期反應與應對**：
- 「太難了，我只要知道公司賺不賺」→ 「只要記住兩本帳的觀念就好：下次看到新聞說某公司『獲利創新高但現金吃緊』，你就是家裡唯一知道那是什麼意思的人」
- 「賒帳也要記？」→ 「對，這是現代會計最聰明的地方：欠的、被欠的都記，才藏不住」

**若對應專欄**：搭配專欄 #7《財務報表三表》服用。

### 練習解答

```
① 借：應付帳款 50,000        貸：現金 50,000
② 借：存貨 60,000            貸：應付帳款 60,000
③ 借：現金 100,000           貸：銷貨收入 150,000
   借：應收帳款 50,000
④ 借：銷貨成本 55,000        貸：存貨 55,000
⑤ 借：現金 30,000            貸：預收收入 30,000   ← 負債，不是收入
⑥ 借：租金費用 20,000        貸：現金 20,000
   借：折舊費用 3,333         貸：累計折舊 3,333
```

**本月淨利** ＝ 150,000 − 55,000 − 20,000 − 3,333 ＝ **71,667**
（⑤ 的 30,000 不算收入，所以不在裡面。）

**月底驗算**（延續上月期末：現金 360,000、設備淨額 196,667、存貨 20,000、應付 50,000、股本 500,000、保留盈餘 26,667）：

| 資產 | 金額 | 負債＋權益 | 金額 |
|---|---|---|---|
| 現金（36 萬 −5 +10 +3 −2 萬） | 420,000 | 應付帳款（5 −5 +6 萬） | 60,000 |
| 應收帳款 | 50,000 | 預收收入 | 30,000 |
| 存貨（2 +6 −5.5 萬） | 25,000 | 股本 | 500,000 |
| 設備淨額（196,667 − 3,333） | 193,334 | 保留盈餘（26,667 + 71,667） | 98,334 |
| **合計** | **688,334** | **合計** | **688,334** ✅ |

**現金變動** ＝ +60,000（36 萬 → 42 萬），**淨利** ＝ 71,667——又不一樣：這次差在「賒銷 5 萬還沒收到」與「預收 3 萬先進來」等時間差。你已經開始用會計的眼睛看世界了。
